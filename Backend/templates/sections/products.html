Products

Product catalog endpoints. All endpoints are public and do not require authentication.

GET /api/v1/products/

Get all products. Public endpoint.

Query Parameters

  • featured - Filter featured products (true/false)
  • category - Filter by category name

Example Request

GET /api/v1/products/?featured=true&category=Electronics

Response (200 OK)

{
  "count": 10,
  "data": [
    {
      "id": 1,
      "name": "Product Name",
      "summary": "Product summary",
      "description": "Full product description",
      "category": "Electronics",
      "cover": "https://example.com/image.jpg",
      "original_price": 149.99,
      "price": 99.99,
      "featured": true,
      "in_stock": true,
      "images": [
        "https://example.com/image1.jpg",
        "https://example.com/image2.jpg"
      ],
      "sizes": ["S", "M", "L"],
      "colors": ["Black", "White"],
      "created_at": "2024-01-01T00:00:00Z"
    }
  ]
}

GET /api/v1/products/search/

Search products by query string. Public endpoint.

Query Parameters

  • q - Search query string (required)

Example Request

GET /api/v1/products/search/?q=laptop

Response (200 OK)

{
  "query": "laptop",
  "count": 5,
  "data": [
    {
      "id": 1,
      "name": "Gaming Laptop",
      "price": 1299.99,
      ...
    }
  ]
}

Response (200 OK - No Query)

{
  "count": 0,
  "data": [],
  "message": "Search query is required"
}
