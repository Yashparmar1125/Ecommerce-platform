Admin

Administrative endpoints for managing products, categories, orders, users, reviews, coupons, and product attributes. All endpoints require admin authentication (superuser credentials).

Authentication

POST /api/v1/admin/login

Admin login. Requires superuser credentials.

Request Body

{
  "username": "admin",
  "password": "adminpassword"
}

Response (200 OK)

{
  "data": {
    "tokens": {
      "access": "eyJ0eXAiOiJKV1QiLCJhbGc...",
      "refresh": "eyJ0eXAiOiJKV1QiLCJhbGc..."
    }
  },
  "message": "Admin login successful"
}

Error Response (401 Unauthorized)

{
  "error": "Invalid credentials",
  "detail": "Invalid username or password"
}

Product Attributes

GET /api/v1/admin/attributes

Get all product attribute values (sizes, colors, etc.). Requires admin authentication.

Query Parameters

  • type - Optional filter for attribute type (size or color)

Reviews

GET /api/v1/admin/reviews

Get all product reviews with optional filtering by product. Requires admin authentication.

POST /api/v1/admin/coupons

Create a new coupon. Requires admin authentication.

Request Body

{
  "code": "NFLAT10",
  "description": "10% off on all products",
  "discount_type": "percentage",
  "discount_value": "10.00",
  "min_purchase_amount": "500.00",
  "max_discount_amount": "200.00",
  "usage_limit": 100,
  "valid_from": "2024-01-01T00:00:00Z",
  "valid_until": "2024-01-31T23:59:59Z"
}

Response (201 Created)

{
  "data": {
    "id": 1,
    "code": "NFLAT10",
    "discount_type": "percentage",
    "discount_value": "10.00",
    "min_purchase_amount": "500.00",
    "max_discount_amount": "200.00",
    "is_active": true
  },
  "message": "Coupon created successfully"
}
